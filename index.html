    How It Works

    Three steps to a transparent result

    EfficientNetB0 — fine-tuned on the BreakHis dataset with patient-level splitting — analyses the tissue and generates a prediction with confidence score.

    Capabilities

    Built for transparency and trust

    EfficientNetB0 Classification

    Trained with focal loss and patient-level data splitting for clinically honest performance evaluation. Achieves 92.4% recall on malignant cases.

    Grad-CAM Heatmaps

    Gradient-weighted Class Activation Maps overlay directly onto the tissue image, showing which cellular regions influenced the classification decision.

    SHAP Explanations

    SHapley Additive exPlanations quantify pixel-level feature contributions, providing a complementary view of model reasoning alongside Grad-CAM.

    Confidence Scoring

    Each prediction includes a calibrated confidence score. Low-confidence predictions are flagged to encourage manual clinical review.

    Multi-Magnification

    Trained and validated across all four BreakHis magnification levels (40X, 100X, 200X, 400X) with consistent recall above 87% at every zoom level.

    Patient-Level Validation

    Strict patient-level data splitting ensures no patient data appears in both training and testing — the clinically correct evaluation methodology.
